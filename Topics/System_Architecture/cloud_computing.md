# Cloud Computing

## Service Models
- **IaaS**: Infrastructure as a Service (AWS EC2, VMs)
- **PaaS**: Platform (Heroku, App Engine)
- **SaaS**: Software (Gmail, Salesforce)

## Deployment Models
- **Public**: AWS, Azure, GCP
- **Private**: On-premises
- **Hybrid**: Mix of public/private
- **Multi-cloud**: Multiple providers

## Benefits
- Scalability, elasticity
- Cost-efficiency (pay-as-you-go)
- Global reach, high availability

## Technologies
- **Containers**: Docker, orchestration (Kubernetes)
- **Serverless**: Lambda functions
- **Microservices**: API gateways, service mesh

## Security
- Shared responsibility model
- Encryption, IAM, compliance
