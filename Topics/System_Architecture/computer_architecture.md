# Computer Architecture

## CPU Architecture
- **Von Neumann**: Stored program concept
- **Harvard**: Separate instruction/data memory
- **RISC vs CISC**: Reduced/Complex instruction sets

## Memory Hierarchy
- **Registers**: Fastest, smallest
- **Cache**: L1/L2/L3, locality
- **RAM**: Main memory
- **Disk/SSD**: Persistent storage

## Parallelism
- **Instruction Level**: Pipelining, superscalar
- **Data Level**: SIMD (vector processing)
- **Thread Level**: Multi-core, hyperthreading

## I/O
- **Buses**: Data/address/control
- **Interrupts**: Hardware/software
- **DMA**: Direct memory access
