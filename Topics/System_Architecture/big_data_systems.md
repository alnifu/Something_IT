# Big Data Systems

## Characteristics (4Vs)
- **Volume**: Massive data
- **Velocity**: High-speed processing
- **Variety**: Structured/unstructured
- **Veracity**: Data quality

## Technologies
- **Hadoop**: HDFS, MapReduce
- **Spark**: In-memory processing
- **NoSQL**: MongoDB, Cassandra
- **Stream Processing**: Kafka, Storm

## Analytics
- **Batch**: Hadoop MapReduce
- **Real-time**: Spark Streaming
- **Machine Learning**: MLlib, TensorFlow on Spark

## Challenges
- Data integration
- Scalability
- Privacy/security
- Skill gaps
