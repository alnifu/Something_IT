# Embedded Systems

## Characteristics
- **Dedicated Functions**: Specific tasks
- **Real-Time**: Hard/soft deadlines
- **Resource Constraints**: Limited CPU, memory, power
- **Reliability**: Fault-tolerant

## Components
- **Microcontrollers**: AVR, ARM Cortex
- **Sensors/Actuators**: Input/output devices
- **RTOS**: Real-time operating systems (FreeRTOS)

## Design
- **Hardware/Software Co-design**: Integrated development
- **Power Management**: Low-power modes
- **Communication**: UART, SPI, I2C, CAN

## Applications
- IoT devices, automotive, medical, industrial control
