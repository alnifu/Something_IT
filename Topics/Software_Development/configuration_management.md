# Software Configuration Management (SCM)

## Concepts
- **Configuration Items**: Code, docs, hardware
- **Baselines**: Stable versions
- **Versions**: Tagged releases
- **Branches**: Feature branches, main

## Tools
- **Git**: Distributed VCS
- **SVN**: Centralized VCS
- **GitHub/GitLab**: Collaboration platforms

## Processes
- **Check-in/Check-out**: Version control
- **Build Management**: Maven, Gradle
- **Release Management**: CI/CD pipelines
- **Change Management**: Peer reviews, approvals

## Best Practices
- Commit often, meaningful messages
- Branching strategies: GitFlow, trunk-based
- Code reviews: Pull requests
- Backup and disaster recovery
