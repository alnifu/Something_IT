# UI/UX Design

## UI (User Interface)
- **Elements**: Buttons, menus, forms, icons
- **Principles**: Consistency, feedback, affordance
- **Tools**: Figma, Sketch, Adobe XD

## UX (User Experience)
- **Process**: Research → Design → Prototype → Test
- **Usability**: Learnability, efficiency, memorability, errors, satisfaction
- **User-Centered Design**: Empathy maps, personas, journey maps

## Design Principles
- **Hierarchy**: Visual importance
- **Accessibility**: WCAG guidelines, screen readers
- **Responsive Design**: Mobile-first, breakpoints
- **Minimalism**: Reduce clutter

## Prototyping
- **Low-fidelity**: Sketches, wireframes
- **High-fidelity**: Interactive prototypes
- **Tools**: InVision, Framer

## Testing
- **Usability Testing**: Observe users
- **A/B Testing**: Compare designs
- **Heatmaps**: User interaction data

## Trends
- **Dark Mode**: Eye strain reduction
- **Voice UI**: Alexa, Siri
- **AR/VR**: Immersive experiences
