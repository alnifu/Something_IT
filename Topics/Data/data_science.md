# Data Science

## Process (CRISP-DM)
- **Business Understanding**: Define problem
- **Data Collection**: APIs, scraping, sensors
- **Data Cleaning**: Impute missing, remove outliers, normalize
- **EDA**: Histograms, scatter plots, correlation matrices
- **Feature Engineering**: Select, transform, create features
- **Modeling**: Train models
- **Evaluation**: Cross-validation, metrics
- **Deployment**: APIs, monitoring, A/B testing

## Statistics
- **Descriptive**: Mean, median, mode, std dev, quartiles
- **Inferential**: t-tests, ANOVA, chi-square, confidence intervals
- **Distributions**: Normal (CLT), Poisson (events), Binomial (trials)
- **Correlation**: Pearson (linear), Spearman (rank), Kendall
- **Regression**: Linear, Multiple, Logistic (odds ratios)

## Machine Learning
- **Supervised**:
  - Regression: Linear, Ridge/Lasso, Polynomial
  - Classification: Logistic, SVM, KNN, Decision Trees, Random Forest, XGBoost
- **Unsupervised**:
  - Clustering: K-Means, Hierarchical, DBSCAN
  - Dimensionality Reduction: PCA, t-SNE, LDA
- **Reinforcement**: Markov Decision Processes, Q-Learning, Deep RL
- **Bias-Variance**: Overfit (high variance), Underfit (high bias)
- **Evaluation**: Accuracy, Precision/Recall/F1, AUC-ROC, Confusion Matrix

## Deep Learning
- **Neural Networks**: Feedforward, CNN (images), RNN/LSTM (sequences)
- **Frameworks**: TensorFlow, PyTorch, Keras
- **Training**: Backprop, Gradient Descent, Adam optimizer
- **Overfitting**: Dropout, Regularization, Early Stopping

## Tools
- **Python**: Pandas (df manipulation), NumPy (arrays), Scikit-learn (ML), TensorFlow (DL)
- **R**: dplyr (pipes), ggplot2 (plots), caret (ML)
- **Jupyter**: Interactive notebooks
- **SQL**: Joins, aggregations, window functions
- **Tableau/Power BI**: Dashboards, storytelling

## Big Data Tools
- **Hadoop**: HDFS (distributed storage), MapReduce (batch processing)
- **Spark**: RDDs, DataFrames, MLlib, streaming
- **Kafka**: Pub-sub messaging, real-time
- **Elasticsearch**: Full-text search, Kibana viz

## Ethics
- **Bias**: Fairness metrics, debiasing techniques
- **Privacy**: Differential privacy, federated learning
- **Transparency**: LIME, SHAP for explainability
- **Accountability**: Model audits, responsible AI
